# samer
samer 是一款基于 Android 平台的大学生社交app
其主要功能有：

1.用户的注册登录（必须验证手机号）
  
  2.用户在说说版块自由发帖，回复，点赞功能
  
  3.即时聊天功能
  
  4.好友查找添加功能
  
  5.发布话题功能
  
  6.查看热门话题功能
  
  7.查看学校所有社团并加入社团功能
  
  8.作为社团管理员可以发布社团活动功能
  
  9.用户查看社团最新活动功能
  
  10.分数查询功能
  
  11.校园公告查看功能
